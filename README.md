# DecisionFlow High-Fidelity Prototype

Production-quality frontend prototype for DecisionFlow, an AI-powered decision intelligence platform.

This project is the high-fidelity evolution of the validated low-fidelity prototype. It preserves the original MVP workflow and improves only the visual implementation, interaction polish, accessibility, and desktop layout.

## Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- shadcn-style UI primitives
- Lucide React
- React Hook Form
- Zod

## Run Locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Available Routes

- `/`
- `/login`
- `/register`
- `/dashboard`
- `/decisions/new`
- `/decisions/upload`
- `/decisions/extraction`
- `/decisions/review`
- `/decisions/workspace`
- `/decisions/recommendation`
- `/decisions/compare`
- `/decisions/library`
- `/profile`
- `/settings`

## Notes

This prototype uses mock data only. It does not include authentication logic, backend services, API routes, or AI calls.
